# plataforma-Versia
Versia é uma plataforma LMS B2B que leva o conceito de "Binge-Learning" para o mundo corporativo. Desenvolvido na Residência FICR em colaboração com a Motiron Technologies, o projeto combina a organização de um sistema de treinamento robusto com a interface amigável e atraente de serviços de streaming como a Netflix.
