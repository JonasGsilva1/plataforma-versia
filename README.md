# Versia - Plataforma de Treinamento Corporativo

## 🚀 Tecnologias

- **Next.js 15** - App Router
- **React 18.3**
- **TypeScript**
- **Tailwind CSS v4**
- **Lucide React** - Ícones
- **Motion** - Animações

## 📦 Instalação

```bash
# Instalar dependências
pnpm install

# Rodar em desenvolvimento
pnpm dev

# Build para produção
pnpm build

# Rodar produção
pnpm start
```
